# 产品需求文档 (PRD): 甘特图视觉升级

## 1. 概述
- **名称**: 甘特图视觉升级 (Gantt Visual Redesign)
- **定位**: 将现有甘特图界面升级为现代化、高保真的 UI 设计，提升用户体验和视觉质感。
- **目标用户**: 项目经理、团队成员。
- **核心问题**: 现有界面较为基础，缺乏视觉层次感和现代感，信息展示不够直观。

## 2. 页面架构
### 页面详细需求
#### 页面：主视图 (Main View)

1.  **全局容器**:
    - 圆角 (16px) 卡片式容器，带有阴影。
    - 内部包含：自定义 Header、Gantt Chart 区域、Footer Legend。

2.  **顶部导航栏 (Header)**:
    - **左侧**:
        - 图标 + 标题 "甘特图" (项目时间线可视化)。
        - 导航箭头 `<` `>` 和当前时间段显示 (如 "2025年9月 - 12月")。
    - **右侧操作区**:
        - **编辑字段**: 蓝色背景，图标+文字。
        - **编辑模式**: 深灰/黑色背景，图标+文字。
        - **导出**: 绿色背景，图标+文字。
        - **导入**: 紫色背景，图标+文字。
        - **全屏**: 图标按钮。

3.  **甘特图主体**:
    - **表格列 (Grid)**:
        - 表头高度增加，文字居中或左对齐（视设计而定）。
        - 任务名称列增加圆形 color badge (紫色)。
    - **时间轴 (Timeline)**:
        - 双层时间刻度：[年/月] - [日]。
        - 周末背景色需区分。
    - **任务条 (Task Bars)**:
        - **核心视觉**: 双色进度展示。
            - 已完成部分 (Progress): 亮绿色渐变 (#22C55E)。
            - 未完成部分: 红色渐变 (#EF4444)。
        - 选中状态: 高亮边框或阴影。
    - **连线 (Links)**:
        - 虚线样式，灰色或浅色。

4.  **底部图例 (Footer)**:
    - 显示 "已完成" (绿块), "未完成" (红块), "依赖关系" (虚线)。

## 3. 样式规范 (Design Tokens)
- **Colors**:
    - Primary Green: `#22C55E` (Completed)
    - Primary Red: `#EF4444` (Incomplete/Overdue)
    - Primary Blue: `#3B82F6` (Edit Fields button)
    - Text: `#1F2937` (Primary), `#6B7280` (Secondary)
    - Grid Line: `#E5E7EB`
- **Typography**:
    - Font Family: `Inter`, sans-serif.

## 4. 验收标准
1.  界面与其 Figma 设计稿 (Node 4:2135) 视觉一致性达到 95% 以上。
2.  所有按钮点击有反馈（hover 效果 + 点击事件）。
3.  甘特图任务条正确显示进度百分比对应的红绿比例。
4.  拖拽任务时，双色条应实时跟随变化。
