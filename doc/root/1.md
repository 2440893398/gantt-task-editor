# 优化
-1.表格中不需要悬浮查看详情功能
-2.运行直接在表格中调整值
-3.本地缓存新增的自定义字段配置和字段排序配置，防止重新进入页面后需要重新配置。最好大量的任务数据也可以本地保存，因为这是一个储前端项目没有后端数据库，要不用户每次进入后都需要重新导入数据很麻烦。
-4.导出导入excel和导出导入json的图标换一下，使其更直观和美观
# 问题
-1.优先级和状态的下拉值没有本地化，切到其他语言后下拉值没有变化
-2.优先级和状态和自定义字段中类型为单选和多选的字段，excel导出模版中没有附带下拉值，这样用户在填写时无法选择，不方便录入。数值类型也要表示这一列只能输入数字，不能输入其他字符。
